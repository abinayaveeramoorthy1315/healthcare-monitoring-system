# 🏥 AI Powered Smart Healthcare Monitoring System

An AI-powered Smart Healthcare Monitoring System that enables secure patient management, intelligent appointment scheduling, emergency response, health monitoring, and real-time notifications using modern web technologies.

---

# 📌 Project Overview

The Smart Healthcare Monitoring System is a full-stack healthcare application developed to improve communication between patients, doctors, and administrators.

The system supports:

- Secure Authentication
- Patient Management
- Doctor Management
- Appointment Scheduling
- AI-Based Health Risk Prediction
- Emergency SOS Alert System
- Ambulance Tracking
- Prescription Management
- Email Notifications
- Real-Time Notifications
- Personal Health Passport

# ⚙️ Installation

## Clone Repository

```bash
git clone https://github.com/yourusername/SmartHealthcareMonitoringSystem.git
```

---

## Backend

```bash
cd backend
mvn clean install
mvn spring-boot:run
```

Runs on:

```
http://localhost:8080
```

---

## Frontend

```bash
cd frontend
npm install
npm run dev
```

Runs on:

```
http://localhost:5173
```

---

# Database

Create a MySQL database named:

```
healthcare_db
```

Update your database credentials in:

```
application.properties
```
